# FlavorWave - Premium Food Delivery UI

A stunning, ultra-modern food delivery web application UI built with React, TypeScript, and Tailwind CSS. This project showcases premium UI/UX design with glassmorphism effects, smooth animations, and a production-ready architecture.

## 🚀 Features

- **Premium UI Design**: Glassmorphism effects, gradient backgrounds, and smooth animations
- **Responsive Layout**: Mobile-first design that works perfectly on all devices
- **Dark/Light Theme**: Toggle between dark and light themes with smooth transitions
- **Interactive Components**: Hover effects, micro-interactions, and smooth page transitions
- **Modern Tech Stack**: React 18, TypeScript, Vite, Tailwind CSS, Framer Motion
- **Authentication Pages**: Login, signup, and forgot password with beautiful forms
- **Landing Page**: Hero section, food categories, popular items, features, testimonials
- **Toast Notifications**: Elegant notification system for user feedback

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React, React Icons
- **Routing**: React Router DOM

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/flavorwave-ui.git
   cd flavorwave-ui
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:4173`

## 🚀 Usage

- **Development**: `npm run dev` - Start development server
- **Build**: `npm run build` - Build for production
- **Preview**: `npm run preview` - Preview production build

## 📁 Project Structure

```
src/
├── components/          # Reusable UI components
│   ├── ui/             # Basic UI components (Button, Input, etc.)
│   ├── Navbar.tsx      # Navigation component
│   ├── Footer.tsx      # Footer component
│   ├── ToastProvider.tsx # Notification system
│   └── Modal.tsx       # Modal component
├── pages/              # Page components
│   ├── Home.tsx        # Landing page
│   ├── Login.tsx       # Login page
│   ├── Signup.tsx      # Signup page
│   └── ForgotPassword.tsx # Password reset page
├── constants/          # Static data
│   └── data.ts         # App data (categories, items, etc.)
├── hooks/              # Custom React hooks
│   └── useScrollProgress.ts # Scroll progress hook
├── utils/              # Utility functions
│   └── clsx.ts         # Class name utility
├── App.tsx             # Main app component
├── main.tsx            # App entry point
└── index.css           # Global styles
```
